Clone the project

```bash
  git clone https://github.com/malinaline/frontend-coding-challenge.git
```

Go to the project directory

```bash
  cd my-react-app
```

Install dependencies

```bash
  npm install
```

Start on Localhost

```bash
  npm start
```


## Tech Stack

**Client:** React, React-Bootstrap, Reqres API

**Server:** Firebase
